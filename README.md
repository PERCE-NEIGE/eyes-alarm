# Eyes Alarm
## what's this?

Web extension used to notice it's time to rest your eyes.

## cycle

every 50 min passed, rest for 10 mins
custom available

## for

- Chrome (`28↑` :ok_hand:) 
- Vivaldi (`1.7↑` :ok_hand:)
- Firefox (`51↑` :ok_hand:)
- Firefox for Android :cold_sweat:
  * BrowserAction API
- Edge :cold_sweat:
  * Alarms API
  * Idle API
  * Notification API
  * extension.getViews
- Opera :japanese_goblin:
